# WF001 - 项目初始化技能

## 技能信息

| 项目 | 内容 |
|------|------|
| 技能ID | WF001 |
| 技能名称 | project-init |
| 触发关键词 | init,项目,初始化,创建工程 |
| 平台支持 | 跨平台 |
| 状态 | 待设计 |

## 触发条件

用户输入包含以下关键词时触发：
- "初始化项目"
- "创建工程"
- "新项目"

## 执行步骤

1. **参数解析**：解析项目名称、类型、模板
2. **创建目录**：创建项目根目录
3. **生成文件**：生成项目结构和配置文件
4. **初始化 git**：可选初始化版本控制
5. **返回结果**：返回项目结构和文件列表

## 输出格式

```json
{
    "status": "success",
    "summary": "项目初始化完成",
    "evidence": [
        {
            "type": "output",
            "content": "项目 my-project 创建成功，包含 12 个文件",
            "title": "初始化结果"
        }
    ],
    "project_path": "/path/to/my-project",
    "files_created": ["README.md", "src/main.c", "Makefile"]
}
```

## 依赖工具

| 工具名称 | 用途 | 最低版本 | 检测方法 |
|---------|------|---------|---------|
| git | 版本控制 | 2.0+ | git --version |

## 安全注意事项

- 确保目标目录不存在
- 项目名称避免使用特殊字符
