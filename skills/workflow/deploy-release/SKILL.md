# WF002 - 部署发布技能

## 技能信息

| 项目 | 内容 |
|------|------|
| 技能ID | WF002 |
| 技能名称 | deploy-release |
| 触发关键词 | deploy,release,部署,发布 |
| 平台支持 | 跨平台 |
| 状态 | 待设计 |

## 触发条件

用户输入包含以下关键词时触发：
- "部署项目"
- "发布版本"
- "deploy"

## 执行步骤

1. **参数解析**：解析目标主机、部署路径、版本号
2. **打包代码**：打包应用程序
3. **传输文件**：将文件传输到目标主机
4. **安装依赖**：安装必要的依赖
5. **启动服务**：启动或重启服务
6. **验证部署**：验证服务是否正常运行
7. **返回结果**：返回部署状态

## 输出格式

```json
{
    "status": "success",
    "summary": "部署完成",
    "evidence": [
        {
            "type": "output",
            "content": "应用已部署到 192.168.1.100:/opt/app",
            "title": "部署结果"
        }
    ],
    "target_host": "192.168.1.100",
    "deployed_version": "v1.0.0"
}
```

## 依赖工具

| 工具名称 | 用途 | 最低版本 | 检测方法 |
|---------|------|---------|---------|
| scp | 文件传输 | - | scp --version |
| ssh | 远程执行 | - | ssh -V |

## 安全注意事项

- 确保目标主机可访问
- 注意敏感信息的安全传输
