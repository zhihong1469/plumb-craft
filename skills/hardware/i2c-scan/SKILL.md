# HW002 - I2C 扫描技能

## 技能信息

| 项目 | 内容 |
|------|------|
| 技能ID | HW002 |
| 技能名称 | i2c-scan |
| 触发关键词 | i2c,scan,扫描,总线,设备 |
| 平台支持 | Linux/裸机 |
| 状态 | 待设计 |

## 触发条件

用户输入包含以下关键词时触发：
- "扫描 I2C"
- "I2C 设备"
- "I2C 总线"

## 执行步骤

1. **参数解析**：解析 I2C 总线编号
2. **权限检查**：检查是否有访问 I2C 设备的权限
3. **扫描总线**：使用 i2cdetect 扫描总线上的设备
4. **解析结果**：提取已连接设备的地址列表
5. **返回结果**：返回扫描结果和设备列表

## 输出格式

```json
{
    "status": "success",
    "summary": "I2C 扫描完成",
    "evidence": [
        {
            "type": "output",
            "content": "总线 0 上发现 2 个设备: 0x50, 0x68",
            "title": "扫描结果"
        }
    ],
    "devices": [
        {"address": "0x50", "description": "EEPROM"},
        {"address": "0x68", "description": "RTC"}
    ]
}
```

- I2C 操作需要相应权限
